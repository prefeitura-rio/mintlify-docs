A IplanRio disponibiliza ferramentas de IA integradas à infraestrutura municipal — com acesso centralizado, autenticado e auditado via Bifrost, nosso proxy de IA.

Não é necessário gerenciar credenciais de cloud ou chaves de API diretamente: basta solicitar acesso e começar a usar.

## Como Funciona o Acesso

```
Você  →  solicita no Discord (#peça-permissão)
       →  recebe Virtual Key por email
       →  roda o script de instalação
       →  usa o OpenCode com múltiplos modelos via Bifrost
```

O **Bifrost** é o proxy centralizado de IA da IplanRio. Ele autentica as requisições, controla o acesso por Virtual Key e faz o roteamento para o modelo correto no Google Vertex AI — sem que você precise configurar nada de GCP.

## Modelos Disponíveis

O Bifrost dá acesso a múltiplos modelos. Escolha o adequado à tarefa para preservar sua cota diária.

| Modelo | Uso recomendado |
|--------|-----------------|
| `anthropic/claude-haiku-4-5` | Padrão — tarefas do dia a dia, explicar código, commits |
| `anthropic/claude-sonnet-4-6` | Features complexas, revisões profundas, refatorações |
| `bifrost-google/gemini-3.5-flash` | Alternativa rápida e econômica |
| `bifrost-google/gemini-3.1-pro-preview` | Raciocínio avançado, arquiteturas complexas |
| `bifrost-google/kimi-k2-thinking` | Raciocínio profundo passo a passo |
| `bifrost-google/glm-5` | Alternativa de baixo custo para tarefas gerais |

Detalhes e otimização de custo no [guia do OpenCode](/ferramentas/opencode-usuario#modelos-disponíveis-e-otimização-de-custo).

## Solicitar Acesso

Acesse o canal **`#peça-permissão`** no Discord da IplanRio com nome, email e justificativa de uso. A aprovação é feita pela equipe de IA e o acesso é provisionado automaticamente.
